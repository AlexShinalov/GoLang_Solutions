# Base64Converter 
## How to use with build:
```bash
go build base64converter.go
.\base64converter.exe encode -i "\hello.txt" -o "\hello64.txt"
```
## Hoe to use without build:
```bash
.\base64converter.exe encode -i "\hello.txt" -o "\hello64.txt"
```
